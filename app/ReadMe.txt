Date : 20/12/2019

Auteur : CLEMENT Alexis

Description : Jeu de stratégie combinatoire abstrait (inspiré du jeu "Puissance 4") se jouant à 2 joueurs tour pas tour. Le but de la partie est de faire un alignement d'au moins 4 pions

Contrôles :
	- Jeu : Clic gauche = Placer un pion
		N = Rejouer une partie (au cas où il y à une égalité (voir Bugs connus))
		Echap = Quitter le jeu

	- GameWin : N = Rejouer une partie
		    Echap = Quitter le jeu

Bugs connus :
	- Si les joueurs jouent et réalisent une égalité (remplir la grille de pions sans réussir à faire un alignement de 4 pions), il ne se passe rien mais il est toujours possible de relancer une partie

Fonctionnalités du jeu : Les 2 joueurs doivent :
	- Réaliser un alignement qui est soit en vertical, en horizontal ou en diagonal d'au moins 4 pions de même couleur
	- Contrer les alignements de l'autre joueur pour ne pas perdre
